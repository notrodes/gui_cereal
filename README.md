# gui_cereal
Toy-application
## Install
1) [Install Rust Cargo](https://www.rust-lang.org/tools/install)
2) [Install GTK-rs Dependencies](https://gtk-rs.org/docs-src/requirements.html)
3) Go to the install directory and run `cargo run` or `cargo run --release`
